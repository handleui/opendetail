---
title: UserMessage
description: Styled row for the user’s question in the assistant thread.
---

**`UserMessage`** renders the user turn in the thread (muted styling, optional layout hooks).

## Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `children` | `ReactNode` | — | Question text or rich content. |
| `className` | `string` | — | Root class. |

Import **`UserMessage`** and **`UserMessageProps`** from **`opendetail-react`**.

---
title: Thread
description: Motion-aware container for user and assistant turns — optional staggered reveal.
---

**`Thread`** wraps the scrollable list of **`UserMessage`** / **`AssistantMessage`** nodes. Set **`animated`** for staggered Motion reveal when the list updates.

## Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `animated` | `boolean` | `false` | Stagger children with Motion. |
| `ariaLabel` | `string` | (default) | Accessible name for the section. |
| `className` | `string` | — | Root class. |

Import **`Thread`** and **`ThreadProps`** from **`opendetail-react`**.

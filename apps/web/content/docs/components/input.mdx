title: Input
description: AssistantInput is the composable prompt field with submit/stop actions, sizing for sidebar and shell layouts, and streaming status integration.
---

**AssistantInput** renders the branded textarea surface, send/stop control, and optional status line. It pairs with `useOpenDetail` request state for pending and streaming modes.

## Sizes

- **`compact`** — default width for narrow rails.
- **`shell`** — wider surface aligned with modal/shell layouts.

## Props

See TypeScript types on `AssistantInputProps` in `opendetail-react` for the full list, including `requestState`, `onSubmit`, `onStop`, `placeholder`, and `showShellUnderlay`.

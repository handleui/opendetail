## `AssistantStatus` (error)

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `variant` | `"error"` | — | **Required** for error styling. |
| `label` | `string` | package default | Visible error copy. |

Use alongside **`Composer`** **`status`** or inside **`AssistantMessage`** **`error`** / **`status="error"`** for richer layouts.

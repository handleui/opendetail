---
title: Status
description: AssistantStatus shows a thinking spinner line or an error line—used under the input or inline in layouts.
---

## Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `variant` | `"thinking" \| "error"` | — | Spinner vs error text. |
| `label` | `string` | variant default | Visible error label; thinking uses a screen-reader label + **`Loader`** icon. |
| `id` | `string` | — | Optional DOM id. |

Thinking variant maps to **`opendetail-status`**; error uses **`opendetail-status--error`**.

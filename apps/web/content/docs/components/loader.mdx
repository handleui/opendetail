---
title: Loader
description: AssistantStatus (thinking variant) shows the thinking spinner line—used under the input or inline while a request is in flight.
---

Import **`AssistantStatus`** from **`opendetail-react`**.

## Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `variant` | `"thinking" \| "error"` | — | Use **`thinking`** for the spinner; **`error`** shows error text (see [**Error**](/components/error)). |
| `label` | `string` | variant default | Screen-reader label; thinking keeps a visible spinner icon. |
| `id` | `string` | — | Optional DOM id. |

Thinking variant maps to **`opendetail-status`**; error uses **`opendetail-status--error`**.

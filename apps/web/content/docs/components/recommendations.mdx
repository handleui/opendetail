---
title: Recommendations
description: AssistantSuggestions and AssistantSuggestion render tappable follow-up prompts below the input or thread.
---

Use **AssistantSuggestions** as a container and **AssistantSuggestion** for each chip-style button. Suggestions are client components and support `onClick` and `disabled`.

## Composition

Wrap multiple `AssistantSuggestion` elements inside `AssistantSuggestions`. Style hooks use `opendetail-assistant-suggestions` and `opendetail-assistant-suggestion` classes from the OpenDetail base stylesheet.

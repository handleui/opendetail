---
title: Pressable
description: opendetail-pressable is a lightweight CSS hook for tactile button feedback—used on suggestion chips, sidebar icon buttons, and other controls.
---

**Not a React component:** import the OpenDetail base stylesheet and add the class **`opendetail-pressable`** to interactive elements (typically **`button`**).

## Behavior

- **`transition`** on transform for a short press scale.
- **`:active:not(:disabled)`** applies **`scale(0.97)`** so disabled controls do not animate.

Pair with your own border / background utilities (see **`AssistantSuggestion`** / sidebar icon buttons in **`opendetail-base.css`**).

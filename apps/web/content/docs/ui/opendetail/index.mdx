---
title: OpenDetail
description: Default assistant UI system — layout regions, React primitives, and Midnight / Signal themes on top of opendetail-base.css.
---

The **OpenDetail** system is the shipped **`opendetail-react`** experience: **`ConversationLayout`**, **`Composer`**, **`AssistantSidebar`**, **`AssistantModal`**, thread primitives, and **`opendetail-base.css`** (with optional **Midnight** / **Signal** theme files). Themes are **palette layers on this system**—swap them without changing React structure.

## Themes

| Theme | Stylesheet | Role |
| --- | --- | --- |
| **Midnight** | `opendetail-midnight.css` | Cooler dark palette on top of base. |
| **Signal** | `opendetail-signal.css` | Warmer accent / palette variant. |

Details: [**Midnight**](/ui/opendetail/themes/midnight), [**Signal**](/ui/opendetail/themes/signal). Concepts: [**Systems and themes**](/ui/systems-and-themes).

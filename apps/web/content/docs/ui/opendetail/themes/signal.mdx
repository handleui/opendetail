**Signal** is a **theme** for the **OpenDetail** system: palette tweaks on top of the base contract; components and layout are unchanged.

Import **`opendetail-base.css`** first, then **`opendetail-signal.css`**. See [**Systems and themes**](/ui/systems-and-themes).

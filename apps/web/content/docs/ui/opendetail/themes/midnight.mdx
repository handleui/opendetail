---
title: Midnight
description: OpenDetail theme — cooler dark palette layered on opendetail-base.css.
---

**Midnight** is a **theme** for the **OpenDetail** system: it overrides semantic color tokens (background, foreground, accent) while keeping the same layout classes and components.

Import **`opendetail-base.css`** first, then **`opendetail-midnight.css`**. See [**Systems and themes**](/ui/systems-and-themes) for the systems vs themes model.

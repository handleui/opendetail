title: Assistant UI

---

This area documents **assistant UI** in three layers:

1. **Foundations** — how **systems** (full layout + semantics) and **themes** (palette layers) fit together, and how CSS is imported.
2. **Hooks** — shared browser APIs (**`useOpenDetail`**, **`createOpenDetailClient`**) that work with any system.
3. **Systems** — each system (starting with **OpenDetail**) has its own **themes** and **components** (`Composer`, `AssistantSidebar`, …).

Start with [**Systems and themes**](/ui/systems-and-themes), [**Hooks overview**](/ui/hooks), or the [**OpenDetail**](/ui/opendetail) system.

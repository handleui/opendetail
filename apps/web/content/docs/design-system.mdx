title: Systems and themes
description: Short overview — full styling guide lives under Assistant UI.
---

OpenDetail splits **systems** (full UI identity: structure, semantics, layout) from **themes** (palette layers on top of one system). The default **OpenDetail** system and **Midnight** / **Signal** themes ship with `opendetail-react` styles.

**Full guide (import paths, primitives, tokens):** [**Assistant UI**](/ui) — [**Systems**](/ui/systems), [**Themes**](/ui/themes), [**Primitives**](/ui/primitives).

**Headless API (same for every system):** [**React**](/docs/react) — `useOpenDetail`, `createOpenDetailClient`.

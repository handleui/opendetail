## setup

Writes the standard project shape for the selected integration mode.

**Writes**

- `opendetail.toml`
- in `self-hosted`, a route at `src/app/api/opendetail/route.ts` by default
- `.opendetail/index.json` unless `--skip-build`

**Important flags**

| Flag | Meaning |
| --- | --- |
| `--integration` | `self-hosted` or `hosted` |
| `--config` | Custom config path |
| `--route` | Custom route path for self-hosted setup |
| `--base-path` | Public source URL base, default `/docs` |
| `--include` | Custom docs glob |
| `--media-include` | Custom media glob |
| `--with-media` | Include `[media]` config |
| `--skip-build` | Skip the initial index build |
| `--force` | Overwrite existing generated files |
| `--interactive` | Force the setup wizard |
| `--no-interactive` | Skip the setup wizard |
| `--cwd` | Run against another working directory |

## build

Reads `opendetail.toml`, parses matched content, writes the local index artifact.

<CopyCommand bun="bunx opendetail build" npm="npx opendetail build" />

Optional flags: `--config`, `--output`, `--cwd`.

## doctor

Checks expected files and env for the selected integration.

**Self-hosted:** config, built index, route file, `OPENAI_API_KEY`.

**Hosted:** config, built index, `OPENDETAIL_ENDPOINT`.

Exits with an error if any check fails.

Use this page when you are **building custom UI**—your layout, your components, your styling. For systems, themes, and primitive styling, see [**Styling**](/docs/design-system); for exports and the streaming hook, see [**React**](/docs/react).

Package APIs: **`opendetail-react`**, **`opendetail-next`**.

## Building blocks

| Approach | Primary APIs |
| --- | --- |
| **Hook + your layout** | [`useOpenDetail`](/docs/react) |
| **HTTP client, no hook** | [`createOpenDetailClient`](/docs/react) |
| **Shell preset** | `AssistantSidebar` with `connection` — [React](/docs/react) |
| **Citations as Next links** | [`renderNextSourceLink`](/docs/next) |
| **Allowlisted doc URLs** | [opendetail-fumadocs](/docs/fumadocs) (optional) |

NDJSON route setup and Node requirements: [opendetail-next](/docs/next). Request shape and stream order: [Get started](/docs/quickstart#integration-modes). Hook, client, and events: [React](/docs/react).

## Package exports

[`opendetail-react`](/docs/react) documents **`useOpenDetail`**, **`createOpenDetailClient`**, **`AssistantMessage`**, **`AssistantSources`**, **`AssistantSidebar`**, and related exports. Take the pieces you need; the full shell is optional.
